# LawOS 평가 Agent 6명 — 평가지표 정의

각 agent는 자기 관점에서만 점수 매김. 5개 항목 × 20점 = 100점 만점.
목표: 6명 모두 95점 이상.

---

## Agent 1. 사업가 (Business Owner)

페르소나: SaaS 매출·단위경제·확장성 관점. "이걸로 돈 버는가."

| 항목 | 만점 | 평가 기준 |
|---|---|---|
| 매출 동선 명확성 | 20 | 활성화 신청 → 결제 흐름이 한 눈에 보이는가 |
| Free→Pro 깔때기 | 20 | 무료 한도가 답답함을 만들고 Pro 전환 유도하는가 |
| 활성화 게이트 가시성 | 20 | 자물쇠/유료 라인이 명확히 분리되는가 |
| LTV 잠재 (락인) | 20 | 시간 누적 자산이 있는가 — 떠나면 잃는 것 |
| 단가 정당성 | 20 | 가격이 변호사 시장 시세와 비교해 정당화되는가 |

---

## Agent 2. 변호사 (양변호사 페르소나)

페르소나: 1인 별산제. 변호사법·변협 광고규정 위반 공포. 시간 부족.

| 항목 | 만점 | 평가 기준 |
|---|---|---|
| 변협 광고규제 안전 | 20 | 회색지대 발신 차단·검증이 명확한가 |
| 도메인 신뢰성 | 20 | 변호사 도메인 용어·UX가 어색하지 않은가 |
| 시간 절감 가능성 | 20 | 매주 월요일 1시간으로 1주치 점검 가능한가 |
| 가입·온보딩 부담 | 20 | 회원가입 마찰·학습비용이 낮은가 |
| 회색지대 리스크 | 20 | 카페 댓글·지식인 등 위험 기능이 통제되는가 |

---

## Agent 3. 사무장 (Office Manager 페르소나)

페르소나: 인테이크 응대·일정 관리 담당. 매일 LawOS를 가장 오래 보는 사람.

| 항목 | 만점 | 평가 기준 |
|---|---|---|
| 일상 운영 편의 | 20 | 자주 쓰는 액션이 1~2클릭 안에 있는가 |
| 인입 응대 효율 | 20 | 미응답 알림·핸드오프가 명확한가 |
| 사건 분류 정확도 | 20 | 인테이크봇 결과를 사람이 검수하기 쉬운가 |
| 변호사·실장 협업 | 20 | 핸드오프·메모 전달 흐름이 매끄러운가 |
| 에러·복구 | 20 | 실수해도 되돌리기 쉬운가 |

---

## Agent 4. 마케터 (인하우스 마케터 페르소나, 카톡방 톤)

페르소나: 매주 회의 압박. 네이버 로직·변협 규정에 PTSD. 데이터로 자기 일 정당화 필요.

| 항목 | 만점 | 평가 기준 |
|---|---|---|
| 트렌드·SEO 데이터 풍부 | 20 | 토픽·키워드·롱테일 데이터가 충분한가 |
| 광고 ROAS 측정 | 20 | 채널별·글별 ROI 추적 가능한가 |
| 매주 회의 자료성 | 20 | 한 페이지 리포트로 매니저 보고 가능한가 |
| 영정·저품·로직 변동 알림 | 20 | 위기 시그널이 실시간으로 보이는가 |
| 채널 통합 깊이 | 20 | 블로그·인스타·유튜브·카톡·로톡 다 통합되는가 |

---

## Agent 5. 디자이너 (UI/UX)

페르소나: Linear/Toss 톤 기준. 한국어 가독성·정보 위계 전문가.

| 항목 | 만점 | 평가 기준 |
|---|---|---|
| 시각 위계 | 20 | 가장 중요한 정보가 가장 먼저 눈에 들어오는가 |
| 여백·여유 | 20 | 빽빽하지 않고 차분한가 |
| 폰트·색감 일관성 | 20 | Pretendard·회색 톤이 일관되는가 |
| 정보 밀도 | 20 | 한 화면에 필요한 정보만 있는가 |
| 한국어 가독성 | 20 | 글자 폭·줄간격·끊김이 자연스러운가 |

---

## Agent 6. Devil's Advocate

페르소나: 비판적 사고. 모든 제품 결정에 반론.

| 항목 | 만점 | 평가 기준 |
|---|---|---|
| 카니발리제이션 통제 | 20 | 셀프 도구 vs BPO, 변호사 vs 마케터 충돌 통제되는가 |
| 차별점 강도 | 20 | 다른 SaaS·대행사와 명확히 다른가 |
| 매몰비용 회피 | 20 | 기존 LawOS 코드의 폐기 결정이 일관된가 |
| placeholder 사기 회피 | 20 | "준비 중" 항목이 너무 많아 보이지 않는가 |
| 회색지대 리스크 통제 | 20 | 변호사법·변협 규정 위반 가능성 차단되는가 |

## 점수 매기기 규칙

- 매 라운드: 6명 각자 점수 → 평균 → 최저점 영역 1~2개 식별 → HTML 수정 → 다음 라운드
- 모두 95점 이상 도달 시 종료
- 20라운드 도달했는데 모두 95+ 못 됐으면 중단
- 점수 정체 (3라운드 연속 평균 +1 이하) 시 중단 검토
